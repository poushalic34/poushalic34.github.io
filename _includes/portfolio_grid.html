PriceMate

Predicts laptop prices using machine learning

PriceMate is a machine learning-powered web application that predicts laptop prices based on various specifications such as processor, RAM, storage, and brand. Built using Python, Flask, and HTML/CSS, it leverages trained machine learning models to provide accurate price estimations. The intuitive web interface allows users to input laptop features and instantly get a predicted price, making it useful for buyers and sellers in the tech market.
